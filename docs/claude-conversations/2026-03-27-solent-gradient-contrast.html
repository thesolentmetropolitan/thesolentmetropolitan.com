Contrast Ratios — WCAG AA

All three text sizes qualify as "large text" under WCAG: bold at 2.4rem (38.4px) and 1.8rem (28.8px) are bold ≥18px; non-bold at 1.8rem (28.8px) is ≥24px. Large text requires 3:1 minimum for AA.

The gradient varies from light gold (#f5c842) to magenta (#e8507a) to soft pink (#f0a0b8). The weakest contrast point is the gold area, as it's the lightest. Ratios below are measured against the lightest point in the gradient.

Alternative — Heading on Gradient, Body in Box

As discussed earlier: large heading directly on gradient, smaller body text in a semi-transparent box. Use this if the enclosure contains mixed content at different sizes.

Welcome to The Solent Metropolitan

We are an independent, not-for-profit media platform championing the Greater Solent region. From the historic roots of Southampton and Portsmouth to our vibrant towns and villages, we are building a distinct global identity for the Solent.

Our "Solent DNA" — Culture, industry Sectors, and Living — forms the triple helix of our mission.

Recommendation: Use #2a2a2a (deep charcoal) for text on this gradient. It passes AA comfortably at every point in the gradient for all three of your text sizes. It's softer than your body text black (#1a1a1a) which feels right — the gradient is warm and vibrant, so a slightly softer dark keeps the overall feel lighter without sacrificing readability. The solent blue (#2c4f6e) looks attractive but fails on the magenta section of the gradient, so avoid it for text that spans the full width. If you want to use solent blue, position the text on the gold/amber side where it passes.

Ready CSS for Classy Paragraphs

Drop this into your enclosure or classy paragraph stylesheet.

CSS for the gradient enclosure paragraph
/* Classy paragraph: warm gradient enclosure */ .paragraph--type--enclosure.gradient-warm { background: linear-gradient(135deg, #f5c842 0%, #f0a030 25%, #e8507a 60%, #f0a0b8 100% ); padding: var(--content-pad, 2rem); } /* Text on gradient — deep charcoal for contrast */ .paragraph--type--enclosure.gradient-warm h1, .paragraph--type--enclosure.gradient-warm h2 { color: #2a2a2a; } .paragraph--type--enclosure.gradient-warm h3, .paragraph--type--enclosure.gradient-warm p { color: #2a2a2a; } /* Override body link styles inside gradient */ .paragraph--type--enclosure.gradient-warm a { color: #2a2a2a; font-weight: 700; text-decoration: underline; text-decoration-color: #2a2a2a; text-underline-offset: 3px; text-decoration-thickness: 2px; } .paragraph--type--enclosure.gradient-warm a:hover { color: #000; } /* Optional: semi-transparent box for smaller body text */ .paragraph--type--enclosure.gradient-warm .content-box { background: rgba(255, 255, 255, 0.92); border-radius: 10px; padding: 2rem; box-shadow: 0 4px 16px rgba(0, 0, 0, 0.06); }
