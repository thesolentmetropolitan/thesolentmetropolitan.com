SVG chevron reuse: The <svg class="svg-defs"> block with the #tvip-down-triangle symbol is already in your page HTML for the menu. CTAs reference it with <use href="#tvip-down-triangle"> and rotate it with transform: rotate(-90deg) in CSS. No dependency issue — if the symbol were ever removed, both menu chevrons and CTA chevrons would break simultaneously, making it immediately obvious. The shared symbol means zero extra file weight.

Background suggestions for the three CTAs together:
• Off-white (#faf9f7) — default body background. Clean, all three colours pop.
• Warm grey (#f5f3f0) — callout bands. Creates a section break.
• Warm gradient — hero/front page. White border needed for separation.
• Solent blue (#2c4f6e) — footer or dark sections. Use the lighter colour variants.
• Pale section tints — #f3e8ff (culture), #dbeafe (sectors), #d1fae5 (living). Use on individual section landing pages to reinforce the section identity. Ghost CTAs work well here as secondary actions.

All contrast ratios pass WCAG AA for normal text (4.5:1 minimum). The ghost variants on light backgrounds are slightly tighter but still pass. On hover, the ghost fills to the section colour with white text, which is even stronger.
